UHL Research Survey Portal

Last updated: 15 June 2026

This portal provides documentation and access for the UHL Research LimeSurvey instance. The public documentation is hosted on GitHub Pages, while the survey application itself is hosted separately on iFastNet.

LimeSurvey requires PHP, MySQL or MariaDB, and server-side processing, so the live survey platform is available at livesurvey.uhl-research.site.

Participant information

Before completing a survey, please read the information provided on the first page of the LimeSurvey questionnaire. That page should explain the purpose of the survey, who is running it, what information is being collected, and how your responses will be used.

General guidance

  • Use a modern browser such as Safari, Chrome, Edge, or Firefox.
  • Complete the survey in one sitting if possible.
  • Do not refresh or close the browser unless instructed.
  • If you receive a unique survey link, do not share it with others.

Privacy and data

Survey responses are collected through the LimeSurvey instance, not through this GitHub Pages documentation site.

This repository should not contain respondent data, participant email lists, exported survey responses, database credentials, SMTP passwords, cPanel credentials, or private configuration files.

Important: All respondent data should remain inside the secured LimeSurvey hosting environment and should be exported, backed up, retained, or deleted according to the project data management plan.

Technical notes

The portal uses a split-hosting setup:

Documentation site

Hosted using GitHub Pages at https://uhl-research.site.

LimeSurvey instance

Hosted on iFastNet at https://livesurvey.uhl-research.site.

Repository

Static files are maintained in the limesurvey-portal GitHub repository.

Recommended repository structure

limesurvey-portal/
├── index.html
├── README.md
└── CNAME

Support

If the survey link does not open, try copying and pasting the direct survey URL into your browser:

https://livesurvey.uhl-research.site

If you are administering the survey, check the LimeSurvey admin panel, iFastNet hosting status, DNS settings, and SSL/HTTPS configuration.
