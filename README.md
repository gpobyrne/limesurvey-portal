# UHL Research Survey Portal

This repository contains the static GitHub Pages landing page for the UHL Research LimeSurvey instance.

The live survey platform is hosted separately on iFastNet because LimeSurvey requires PHP, MySQL/MariaDB, writable server directories, and server-side execution. 
GitHub Pages is used only for the static public-facing landing page and documentation.

## Live links

- Survey portal / documentation site: `https://gpobyrne.github.io/limesurvey-portal/`
- LimeSurvey instance: `https://livesurvey.uhl-research.site`

If a custom domain is configured for this GitHub Pages site, the portal may also be available at:

```text
https://uhl-research.site
```
